# x127

127xx — ローカル開発ツールのためのポート台帳。
ホストで LISTEN 中のポートを一覧し、任意のポートに名前・メモを付けて管理する。

名前の由来: `127.0.0.1` + `xx`。ポート `12700`〜`12799` を自分の管理領域として使う。
`12700` は x127 自身の Web UI、`12701`〜`12799` は将来のランチャー機能の自動割当プール。

> **Status**: v0.1 実装中

## 予定している使い方

```sh
x127 serve    # バックグラウンド起動 → http://127.0.0.1:12700
x127 status   # 稼働確認
x127 stop     # 停止
```

- Go シングルバイナリ + 埋め込み Svelte UI
- 設定・データ: `~/.config/x127/`(registry.json / x127.pid / x127.log)
- 対応 OS: macOS / Linux
